---
name: Greeting message style rules
description: User's strict preferences for Otto welcome greeting tone — no time refs, no cringe, no competitive language
type: feedback
---

Welcome greetings must be quirky little sayings that align with the time of day WITHOUT explicitly referencing the time.

**Why:** User finds explicit time references ("it's 11 AM, almost lunch") annoying and motivational/competitive language cringy.

**Rules:**
- NEVER say the time ("it's 5 PM", "almost lunch", "3 AM")
- NEVER use competitive/better-than-others language ("while they sleep", "your competitors are dreaming", "playing chess while they play checkers")
- NEVER use motivational poster energy ("fortune favors", "keep grinding", "outlast them all", "quitting time is for quitters")
- DO use idioms, clever wordplay, truck/sales humor, self-aware AI wit

**Gold standard examples:**
- "Burning the midnight oil?"
- "Truck data by moonlight. Very noir."
- "Not done 'til the fat lady signs the PO."
